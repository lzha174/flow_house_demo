Introduction

CSimPy is a lightweight C++20 coroutine-based discrete event simulation engine, inspired by SimPy. It is designed to model asynchronous workflows and resource contention efficiently, offering better control and performance than Python-based systems in performance-critical environments.

The framework enables you to define coroutine-based processes that wait for events, interact with resources, and yield control based on simulation time.

Key Features

  • Native C++20 coroutine support
  • Efficient event queue for large-scale simulation
  • Support for resource acquisition and release semantics
  • Customizable simulation clock and event scheduling
