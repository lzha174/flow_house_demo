Introduction

Flow House is a powerful discrete event simulation package designed to optimize complex process flows. It is an improvement over my earlier project, HealthCare Acute Flow.

Major Improvements

  • Multi-resource handling: Jobs can require multiple staff members with different skills.
  • Smarter job dispatch: Enhanced logic for scheduling compared to the old one-staff-per-job rule.
  • Petri net modeling with Snakes: Easier and more intuitive workflow design than PM4Py.

Built with Python, SimPy, and Streamlit, Flow House uses JSON job profiles and dashboards tailored for researchers and engineers.
