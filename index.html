Introduction

Flow House is a powerful discrete event simulation package designed to optimize complex process flows involving concurrent events. It offers precise modeling of real-world processes where multiple tasks can occur simultaneously, improving efficiency and decision-making.

This framework is an enhancement of my earlier project, HealthCare Acute Flow.

Major Improvements

  • Multi-resource handling: Jobs can now request multiple staff members with different skills (previously limited to one job per staff).
  • Smarter job dispatch: Updated dispatch logic to support the complexity of multi-resource handling.
  • Petri net modeling with Snakes: Easier and more intuitive workflow design than PM4Py.

What You Can Optimize

Roster Optimization

Optimize staffing schedules to ensure coverage, balance workload, and respect shift constraints.

Job Arrival Optimization

Control and schedule incoming tasks to reduce queueing and improve flow efficiency.

Workflow Strategy Optimization

Test and improve different process paths using simulation feedback.

Stress Testing

Use the synthetic data generator to simulate high-load scenarios and evaluate system resilience.
